# TDA - Hash

Hash abierto implementado en lenguaje C para la materia Algoritmos y programacion II (FIUBA)



## Compilacion

* Para realizar la compilacion de los tests:

`gcc -Wall -Wextra -Werror  -ggdb -o0 *.c -o nombre_ejecutable`

* Para ejecutarlos: 

`./nombre_ejecutable`
